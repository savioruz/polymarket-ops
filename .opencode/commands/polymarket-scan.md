# /polymarket-scan — Scan leaderboard for top traders

Load modes/scan.md and execute the full leaderboard scan.

This will:
1. Fetch top 20 traders by monthly PnL
2. Score each one using the A-F system
3. Recommend wallets to add/remove from watchlist
4. Identify any market consensus signals

Output a markdown report to reports/scan-{DATE}.md
